# BLDC drivers SPbUnited

## Цветовая индикация

ОК - горит зеленый, периодически на мгновение выключающийся

ПРЕДУПРЕЖДЕНИЕ ПО ТЕМПЕРАТУРЕ - попеременно горит зеленый и желтый. Общий период в секундах говорит о типе предупреждения:

- 1сек - предупреждение о температуре платы
- 2сек - предупреждение о температуре мотора
- 3сек - предупреждение о температуре платы и мотора

ОШИБКА ПО ТЕМПЕРАТУРЕ - попеременно горит желтый и красный. Общий период в секундах говорит о типе ошибки:

- 1сек - критическая температура платы
- 2сек - критическая температура мотора
- 4сек - ошибка инициализации мотора
- 8сек - ошибка калибровки мотора
- 16сек - ошибка мотора

Другое время складывается из представленных как в двоичной системе, напрмер 3сек это критическая температура платы и мотора.
